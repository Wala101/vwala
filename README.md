# vWALA Platform

Plataforma de previsões construída em Polygon, com economia própria em vWALA, swap interno, stake integrado e arquitetura de treasury para sustentação do ecossistema.

## Visão

O projeto vWALA nasce com foco em:

- previsões on-chain em Polygon
- economia própria com token vWALA
- swap interno sem depender de pool pública no lançamento
- stake integrado à base de liquidez
- treasury operacional para cobertura e reciclagem do sistema
- bootstrap inicial sem poderes administrativos posteriores

## Princípios

- Rede base: Polygon
- Preço interno inicial: 1 POL = 1 vWALA
- Admin apenas para inicialização
- Sem pausa, saque manual ou alteração posterior da lógica principal
- Liquidez e sustentabilidade como pilares do ecossistema

## Estrutura inicial

- `index.html` → entrada principal
- `src/index.js` → render da home
- `src/index.css` → identidade visual da plataforma
